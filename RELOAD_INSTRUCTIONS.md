# How to Reload the App on Your Phone

The screens have been updated! To see the changes, you need to reload the app:

## Method 1: Shake to Reload (Easiest)
1. **Shake your phone** while the Expo Go app is open
2. A menu will appear
3. Tap **"Reload"**
4. The app will refresh with all the new screens

## Method 2: From Terminal
1. In the terminal where `npm start` is running
2. Press **`r`** (lowercase r)
3. This will reload the app on your phone

## Method 3: Restart Expo Server
1. In the terminal, press **Ctrl+C** to stop the server
2. Run `npm start` again
3. Scan the QR code again on your phone

## If You Still See "Coming soon..." Text:
This means the app hasn't reloaded properly. Try:
1. Close Expo Go completely (swipe it away from recent apps)
2. Reopen Expo Go
3. Scan the QR code again

## Troubleshooting:
- **Red screen error?** Check the terminal for error messages
- **White screen?** The app is loading, wait a few seconds
- **Old screens showing?** Make sure you reloaded using one of the methods above
